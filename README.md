### Included in this Ionic React Template/UI
* View Recipes by pre-defined categories
* Search any recipe (API)
* Add to bookmarks
* View nutritional values and ingredients
* Global state management with Pullstate
* SASS Modules
* Ionic CSS utilities

### To run

```javascript
npm install
ionic serve
```
